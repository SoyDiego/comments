# Guestbook

A simple guestbook, that's it :).

## Technologies

-   React + Redux
-   Styled Components
-   Firebase

## You can

-   Register / Login users.
-   Protected routes.
-   Add comments.
-   Delete comments (only the user creator delete your own comment).
-   Like / Dislike comments.
-   Count likes.
